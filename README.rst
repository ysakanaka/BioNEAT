Installation
============



Develop
-----------

Requirements
~~~~~~~~~~~~

For building:

* Java 8

For running:

* Java 8

Version info
~~~~~~~~~~~~

::

  ./gradlew -v

Create Eclipse project
~~~~~~~~~~~~~~~~~~~~~~

::

  ./gradlew eclipse

Create IntelliJ project
~~~~~~~~~~~~~~~~~~~~~~~

::

  ./gradlew idea


Run the Demo
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

main class of use.processing.targets.RDBottomNicolas


Production
-----------

Requirements
~~~~~~~~~~~~

For building:

* Java 8

For running:

* Java 8

Version info
~~~~~~~~~~~~

::

  ./gradlew -v
  

Compiling
~~~~~~~~~

::

  ./gradlew dist
  
Run the Demo
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

::
  
  ./build/dist/start.sh use.processing.targets.RDBottomNicolas
